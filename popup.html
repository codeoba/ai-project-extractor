AI Project Extractor

This extension extracts project folder structures and code blocks from AI chats and exports them as a single ZIP file.

Supported Sites:
  • Gemini
  • Claude
  • DeepSeek
  • ChatGPT

Open a supported AI chat, look for code outputs, and click the floating 📁 button in the bottom-right corner to extract.
